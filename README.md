# Number Battle Rush 🎮

하이퍼캐주얼 숫자 배틀 러시 게임 — 오토런 + 레인 스와이프 + 숫자 타일 수집 + 보스 배틀!

## 🎯 게임 컨셉

**Count Masters / Number Hunter** 스타일의 웹 게임입니다.

- **오토런**: 플레이어가 자동으로 달립니다
- **3개 레인**: 좌/중/우 레인 사이를 이동
- **숫자 타일 수집**: `+N`, `×N`, `−N` 타일로 내 숫자를 키움
- **보스 배틀**: 25초 후 보스 등장! 내 숫자 ≥ 보스 HP → 승리!

## 🕹️ 조작법

| 방법 | 동작 |
|------|------|
| `←` `→` 화살표 키 | 레인 변경 |
| 화면 탭 | 탭한 위치의 레인으로 이동 |
| 좌우 스와이프 | 레인 변경 |

## 🚀 플레이 방법

### 방법 1: 파일 직접 열기
`index.html` 파일을 브라우저로 드래그&드롭하거나 더블클릭합니다.

### 방법 2: 로컬 서버로 실행
```bash
# Python 3
python3 -m http.server 8000

# Node.js (npx)
npx serve .
```
브라우저에서 `http://localhost:8000` 접속

## 📐 게임 규칙

| 타일 | 효과 |
|------|------|
| `+N` (초록) | 숫자 N 증가 |
| `×N` (노랑) | 숫자 N배 증가 |
| `−N` (빨강) | 숫자 N 감소 (최소 1) |

- 게임 시간: **25초**
- 시간이 끝나면 **보스 등장**
- 내 숫자 **≥** 보스 HP → **승리! ⭐⭐⭐**
- 내 숫자 **<** 보스 HP → **패배...**

## 🛠️ 기술 스택

- **HTML5 Canvas** — 순수 JavaScript, 외부 라이브러리 없음
- 브라우저 직접 실행 가능 (에뮬레이터 불필요)
- 모바일 터치 지원
